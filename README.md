# Nitro Discord Checker
## Made By Trephy

Simple .txt file based nitro gift checker

3.76s/100 lines <br>
34.36s/1000 lines

## Getting Started 

1. Clone The Repo
2. Add your .txt file in '/txt'
3. Run <code> go run main.go txt/FILENAME.txt </code>

### Prerequisites

Have Go installed: https://go.dev/dl/
